# AI工具集

一个现代化的AI工具导航平台，帮助用户快速发现和使用各类AI工具。

## 项目结构

```
Navigation/
├── css/
│   └── style.css          # 样式文件
├── js/
│   ├── main.js            # 主要JavaScript逻辑
│   └── websites.js        # 网站数据
├── index.html             # 主页面
└── README.md              # 项目说明文档
```

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- Bootstrap 5 (响应式框架)
- Font Awesome (图标库)

## 功能特点

1. **AI工具分类展示**：按不同类别展示AI工具，如大语言模型、图像生成、音频处理等
2. **多搜索引擎支持**：集成百度、必应、谷歌、知乎等多个搜索引擎
3. **快捷导航**：提供常用AI工具的快捷访问
4. **AI新闻资讯**：展示最新AI领域动态和新闻
5. **响应式设计**：适配PC端和移动端，提供良好的用户体验
6. **美观的UI**：使用Bootstrap 5和Font Awesome，提供现代化的界面设计

## 使用方法

1. 直接打开`index.html`文件即可使用
2. 点击左侧分类菜单可以筛选不同类别的网站
3. 使用顶部搜索框可以快速查找需要的网站

## 自定义网站

如需添加或修改网站，请编辑`js/websites.js`文件，按照以下格式添加网站信息：

```javascript
{
    id: 数字ID,
    name: "网站名称",
    url: "网站URL",
    description: "网站描述",
    category: "分类名称",
    icon: "图标类名"
}
```
